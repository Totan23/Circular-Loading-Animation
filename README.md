# Circular Loading Animation

Esta es una app de iOS que muestra una animación circular de carga con un efecto visual bastante chulo. Cuando la abres, verás un círculo en el centro de la pantalla que pulsa constantemente. Si tocas la pantalla, la app comienza a descargar un archivo y el círculo se va llenando poco a poco mostrando el progreso de la descarga en tiempo real con el porcentaje en el medio. Todo está hecho con Swift usando Core Animation para las animaciones y URLSession para manejar las descargas. Es básicamente un ejemplo de cómo crear indicadores de progreso circulares animados de forma suave y moderna.
